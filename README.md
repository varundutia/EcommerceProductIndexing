# E-commerce Product Indexing

A Java application that explores efficient product indexing and search techniques for e-commerce catalogues using custom data structures and indexing strategies.

The project demonstrates how products can be organised to support fast lookup, filtering, and retrieval operations.

---

## Features

- Product indexing
- Fast product lookup
- Search and retrieval
- Custom data structures
- Performance analysis
- CSV dataset processing
- Command-line interface

---

## Tech Stack

- Java
- Object-Oriented Programming
- Data Structures
- Indexing Algorithms
- CSV Processing

---

## Getting Started

Clone the repository

```bash
git clone https://github.com/varundutia/EcommerceProductIndexing.git
```

Compile

```bash
javac -d out $(find src -name "*.java")
```

Run

```bash
java -cp out Main
```
